NeuralChat

AI Chat Assistant · 2024

NeuralChat is a full-stack AI chat application supporting multiple language models (GPT-4, Mistral, Llama 3). It features a Retrieval-Augmented Generation (RAG) pipeline that lets users upload their own documents and query them via natural language. Conversations are stored with a semantic search index for context-aware replies.

Key Features

  • Multi-model support (GPT-4o, Mistral, Llama 3 via Ollama)
  • RAG pipeline with FAISS vector store for custom document Q&A
  • Persistent conversation history with semantic search
  • Streaming responses with Server-Sent Events
  • User authentication with JWT and session management
  • Dockerized deployment with nginx reverse proxy

DataPulse Dashboard

Analytics Platform · 2023

DataPulse is a real-time SaaS analytics platform for monitoring business metrics. It provides live dashboards, customisable KPI cards, trend analysis and automated PDF/email reports. Data is ingested via a lightweight event API and processed with a streaming pipeline, making it suitable for high-throughput applications.

Key Features

  • Real-time data streaming with WebSockets
  • Interactive D3.js charts (line, bar, scatter, heatmap)
  • Custom KPI builder with drag-and-drop layout editor
  • Automated scheduled reports delivered via email (PDF)
  • Role-based access control for teams
  • REST API for external data ingestion

EcoTrack

Mobile App · Flutter · 2024

EcoTrack helps individuals measure and reduce their carbon footprint through daily habit tracking, AI-powered suggestions and community challenges. An onboarding questionnaire builds a personalised eco-profile; the ML model estimates emissions from transport, food and energy usage and suggests evidence-based improvements with a gamified reward system.

Key Features

  • Carbon footprint calculator covering transport, diet and home energy
  • ML model for accurate emission estimation and trend forecasting
  • Gamification: streaks, badges, weekly challenges and leaderboards
  • Community groups and challenges with friends
  • Offline-first architecture with background sync
  • Push notifications for daily reminders and milestones

DevFlow CLI

CLI Tool · Go · Open Source · 2023

DevFlow is a developer-experience CLI tool written in Go that eliminates boilerplate by scaffolding production-ready project structures, generating Dockerfiles, CI/CD pipelines and pre-commit hooks in seconds. It supports multiple templates (Go API, Python service, Flutter app) and cloud providers (GCP, AWS, Fly.io).

Key Features

  • Project scaffolding with language-specific templates
  • Automatic Dockerfile generation (multi-stage, optimised image sizes)
  • GitHub Actions / GitLab CI pipeline generation
  • Interactive devflow dev with hot-reload via Air or Watchexec
  • One-command deploys to GCP Cloud Run, AWS ECS or Fly.io
  • Plugin system for extending commands with custom scripts
